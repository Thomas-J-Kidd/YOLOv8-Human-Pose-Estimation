Pose Estimation and Tracking Project

Using YOLO and OpenCV

Project Overview

In this project, you will implement a Python program to perform pose estimation and object tracking using YOLO and OpenCV.

Step 1: Environment Setup

Required Libraries:

  • opencv-python
  • ultralytics
  • argparse
  • numpy

YOLOv11 Pose Estimation Model

Download YOLOv11n-pose Model

  • YOLOv11n-pose is a lightweight pose estimation model.
  • Download the model from Ultralytics: Download YOLOv11n-pose
from ultralytics import YOLO
model = YOLO('yolov11n-pose.pt')

Step 2: Keypoint Detection

Keypoints:

  • Detect key body points such as:

    • Nose
    • Eyes, Ears
    • Shoulders, Elbows, Wrists
    • Hips, Knees, Ankles

  • Keypoints are labeled in the image or video.

Visualization:

Keypoints should be displayed as circles, and bounding boxes should contain IDs and confidence scores.

Keypoint Detection Example

Code Example:

import cv2
from ultralytics import YOLO

# Load YOLOv11n-pose model
model = YOLO('yolov11n-pose.pt')

# Load an image
img = cv2.imread('people_image.jpg')

# Perform inference
results = model.predict(img)

# Plot results with keypoints
results[0].plot()
